# OctoDex
Github's Octocat + Rolodex = OctoDex
* Easily create and share a virtual business card using your Github information

## Front-End
* Angular - For MVC 
* Angular Material - For nice Design 

## Back-End 
* Node.js - Server
* MongoDb - Database
* Passport.js -  Authentication middleware for Node.js.
* Github's Oauth2
